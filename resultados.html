Prueba 01

ANPEIX 100 PLUS SP

Carne de centollón cocida

Prueba orientada a evaluar comportamiento de rendimiento, reducción de pérdida de líquidos propios y estabilidad de textura en una matriz de alto valor comercial. El bloque queda preparado para reemplazar este texto por datos reales de proceso, condiciones de aplicación, dosis, temperatura y lectura final.

24–48 hTiempo de implementación

Prueba 02

ANPEX GLASS

Producto congelado glaseado

Evaluación visual y operacional de glaseo protector para mejorar apariencia, reducir deshidratación superficial, disminuir pegado y estabilizar el comportamiento del producto durante congelación, almacenamiento y descongelación.

+1–3%Retención esperada de glaseo
1 jornadaAjuste operativo inicial
VisualBrillo, cobertura y descongelación
Prueba 03

PhageGuard Listex

Superficie de producto listo para consumo

Prueba diseñada para evaluar aplicación dirigida sobre patógeno objetivo, bajo comparación control/tratado y registro de condiciones de uso. Este bloque permite documentar matriz, punto de aplicación, dosis, tiempo de contacto y resultado microbiológico.

1–2 logReducción referencial esperada
24–48 hLectura microbiológica
PACRegistro QA y trazabilidad
Prueba 04

TQI BLANTEX / Línea específica

Moluscos, cefalópodos o producto marino procesado

Bloque adicional preparado para documentar una prueba de textura, color, estabilidad o comportamiento de producto en proceso. Puede usarse para registrar una evaluación de matriz distinta, una mejora visual, una comparación de rendimiento o una aplicación específica solicitada por planta.
